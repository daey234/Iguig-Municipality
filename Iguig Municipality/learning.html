MY LEARNING INSIGHTS

In our web development course, we started by delving into the basics of HTML (HyperText Markup Language) and CSS (Cascading Style Sheets), both of which are fundamental for creating and designing web pages. HTML provides the essential structure of a webpage, functioning as its skeleton. On the styling side, CSS allowed us to enhance the visual appeal and layout of our HTML structures. We learned about selectors, which target specific HTML elements to apply styles, including type selectors, class selectors, and ID selectors. Understanding properties and values in CSS was crucial.

MY LEARNING INSIGHTS

Applying our knowledge practically, we learned to build simple webpages by combining HTML and CSS. This process involved starting with the HTML structure and then applying CSS to style it, allowing us to see the immediate impact of our code. Utilizing browser developer tools, such as Chrome DevTools, enabled us to inspect elements and debug CSS, providing a hands-on approach to troubleshooting and refining our work. We also discovered the importance of experimenting with different HTML elements and CSS styles, continuously iterating and observing changes to improve our designs.
